# Laboratorium 03 - Integracja systemów informatycznych

**Cel laboratorium: Rozbudowa aplikacji o mechanizmy integracji z zewnętrznymi API, przetwarzanie danych JSON oraz ich wizualizacja w aplikacji webowej.**

## Wymagania:

- Działająca integracja z dwoma zewnętrznymi API.
- Udokumentowany proces obróbki i filtrowania danych.
- Widoczna wizualizacja danych na stronie.
- Własny endpoint API serwujący zagregowane dane zewnętrzne.
- Poprawna historia commitów na dedykowanej gałęzi.
